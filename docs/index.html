How it works

1

srun sends your PID to the server

Requests N GPUs. Blocks until they're available.

2

Server checks nvidia-smi

Polls every second. FIFO queue â€” first come, first served.

3

GPUs assigned â†’ your command runs

CUDA_VISIBLE_DEVICES is set, then exec replaces srun with your command.

4

Process exits â†’ GPUs released

Server monitors your PID. When it's done, GPUs go back to the pool.
